                                                                 Array.prototype.includes()
 
 1. Fathima Saja S :
        The includes() method checks a certain value is present in array or not. If the specific value is found, it returns true else returns false. 
        
 2.Babisha V:
       It is used to check whether the certain value is present in an array.
       It returns true or false.
       Syntax:
         includes(searchElement)
         includes(searchElement, fromIndex)
       The index value and the values after the index value is checked for search element,
       Examples:
          const array1 = [1, 2, 3];
          console.log(array1.includes(2));// true
       Condition:
           fromIndex > array length (array is not searched)// false
           computed index=length+index (complete array is checked)
           fromIndex < 0 (fromIndex+array length is used)
           fromIndex>= array length// false
       Example:
           const array1 = [1, 2, 3];//array.length=3
           console.log(array1.includes(2,-2));// true
           //(3+(-2))=1
       



                                                                Array.prototype.indexOf()
                                                              
 1. Fathima Saja S:
              The indexOf() method gives the first index of the search element present in the array. If the element is not present in the array, it returns -1. 
              When the element is present in multiple times in an array, it can be searched from a particular index by indexOf(searchElement, fromIndex). 

 2. Babisha V:
             This method is used to get the first index of the search element.If the search element is not present in an array,it returns -1.
             If present return the index value.
        Syntax:     
             indexOf(searchElement)
             indexOf(searchElement, fromIndex)// fromIndex:default 0





                                                                 Array.isArray()
                                                                 
 1. Fathima Saja S:
                 Array.isArray() method checks if the passed value is present or not. If the passed value is numbers, strings, null, boolean values or undefined values not inside an array will return false.
                 
 2.Babisha V:
               It is used to check whether the passed value is an array or not.
               If the passed value is an array return true else return false.
        Syntax:
           Array.isArray(value)
